Dream Wedding Planner is a responsive and elegant wedding planning website designed to showcase wedding services and attract potential clients. The website provides information about luxury wedding planning services, pricing packages, wedding galleries, and consultation options in a single-page layout. It includes a navigation bar for easy access to different sections, a visually appealing hero section, service cards highlighting wedding offerings, package plans for different budgets, a gallery displaying wedding moments, and a contact form for customer inquiries.
   
   
            The website is built using HTML for structure, CSS for styling and responsiveness, and JavaScript for basic interactivity such as smooth navigation and form submission alerts. The overall design focuses on creating a premium and user-friendly experience for couples planning their dream wedding. Similar wedding-planning websites commonly organize services, packages, galleries, and contact information in one place to help couples easily explore offerings and make inquiries.

1.Smooth navigation bar
2.Attractive hero/banner section
3.Wedding service showcase
4.Pricing and package plans
5.Interactive image gallery
6.Contact and inquiry form
7.Responsive and modern design
8.Hover effects and animations for better user experience
